# Part1

Cybersecurity Awareness Chatbot - Part 1

Overview

This is a C# console application designed to act as an interactive Cybersecurity Awareness Chatbot. It provides users with an engaging, text-based interface to learn about online safety, complete with visual enhancements and an audio greeting.

Features

Interactive Console UI: Uses custom typing effects and colored text to simulate a real terminal interface.

Audio Integration: Plays a custom .wav audio greeting when the application launches using System.Media.SoundPlayer.

Personalized Experience: Asks for the user's name and customizes the chat experience.

Modular Code Structure: Organized into separate classes (Program.cs, Chatbot.cs, UIHelper.cs) for clean, maintainable code.

Prerequisites

Visual Studio 2022 (or newer)

.NET SDK (Targeting Windows, as SoundPlayer is a Windows-specific feature)

How to Run the Application

Clone this repository to your local machine.

Open the solution file in Visual Studio.

Ensure that the greeting.wav file in the project has its Copy to Output Directory property set to Copy if newer.

Press the Start (Green Play button) at the top of Visual Studio to build and run the application.
